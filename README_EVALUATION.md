# Hermetic Labs - NHS-SBS CQ Portal (Technical Demonstrator)

## 1. Zero-Trust, Zero-Install Architecture
This executable is a fully portable, zero-install application. 
It requires absolutely no network connectivity. It makes zero outbound requests. All data processing, including Large Language Model (LLM) inference, is handled entirely locally on the host machine's hardware using the bundled 4GB weights.
